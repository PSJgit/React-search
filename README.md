# REACT SEARCH

## What is it?

Tiny search function for lists and the like, in React

## Demo

Go here: [React-search](https://psjgit.github.io/react-search/)

## Built With

* Javascript
* React
* Babel
* Webpack

## Commands

To install the project, run:

```
npm i
```

To run in development mode:

```
npm run dev
```

And to create a production version:

```
npm run build
```
